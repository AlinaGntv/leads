# Лиды

Мини-приложение для личного трекинга рассылок лидам.

## Как работает

- Цель дня: 45 касаний.
- Цель месяца: 1000 касаний.
- Каждый клик по кружочку фиксирует прогресс за текущий день.
- Данные сохраняются в `localStorage` браузера.
- Для нового месяца создается отдельное хранилище.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите в него файлы `index.html`, `styles.css`, `script.js` и `README.md`.
3. Откройте настройки репозитория: `Settings -> Pages`.
4. Включите публикацию из ветки `main`, папка `/root`.

Готово: приложение будет работать без сервера и базы данных.
